# LangChain: From RAG to Agentic AI

This repository focuses on **practical implementation of LangChain**, covering everything from core building blocks to advanced concepts like **Retrieval-Augmented Generation (RAG)** and **Agentic AI workflows**.

The goal is to move beyond theory and build a solid understanding of how **LLM-powered systems** are designed, structured, and optimized in real-world scenarios.

---

##  What This Repo Covers

###  Foundations
- LangChain architecture and core components
- Prompt templates and output parsers
- Chains and execution flow

###  Vector Databases & Retrieval
- Embeddings using HuggingFace models
- FAISS / Chroma vector stores
- Similarity search and document retrieval

###  Retrieval-Augmented Generation (RAG)
- End-to-end RAG pipeline implementation
- Context-aware prompting
- Multi-query retrieval strategies
- Reducing hallucinations and improving accuracy

###  Tool Calling
- Creating and integrating custom tools
- Tool binding with LLMs
- Understanding tool invocation workflows
- Sequential execution vs controlled orchestration

###  Agentic AI (Introduction)
- Fundamentals of agent-based systems
- Decision-making using LLMs
- Tool-using agents and dynamic workflows
